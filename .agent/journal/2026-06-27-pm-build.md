## Game Concept
Card Concentration / Pair Matching Puzzle — flip cards to find matching pairs across 8 game modes and 3 difficulty levels.

## Features Implemented
- 8 game modes: Classic, Timed, Limited Moves, Speed Match, Zen, Daily Challenge, Endless, Blind Recall
- 3 difficulties: Easy (4×3), Medium (4×4), Hard (5×4)
- 40 achievements with localStorage persistence
- 8 card skins with gameplay-gated unlocks
- 5 holodeck themes
- XP/Level progression (50 levels)
- Combo scoring (x1-x10 multiplier)
- Seeded daily challenge (mulberry32 PRNG)
- Procedural audio: 12+ SFX + ambient drone
- 150-particle pool with burst effects
- XR controller support (laser pointer + trigger to flip)
- 14 PanelUI panels (zero DOM UI)
- Holodeck environment with floor/ceiling grid, floating decorations, ambient particles
- Full game stats tracking
- GitHub Pages deployment
